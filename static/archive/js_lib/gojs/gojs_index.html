Watermark removal in GOJS

Find go.js and go-debug.js and search in js file 7eba17a4ca3b1a8346 will have the following code: 

a.cr=b.W[Za("7eba17a4ca3b1a8346")][Za("78a118b7")](b.W,Ak,4,4);

Replaced with 

a.cr=function(){return true;};

Run again to see that the watermark has been removed.

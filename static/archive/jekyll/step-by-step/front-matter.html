Front Matter

TOC Toggle Commentary

Front matter is a snippet of YAML
placed between two triple-dashed lines at the start of a file.

You must include front matter on the page for Jekyll to process any Liquid tags on it.

You can use front matter to set variables for the page:

---
my_number: 5
---

You can call front matter variables in Liquid using the page variable.
For example, to output the value of the my_number variable above: 

{{ page.my_number }}

To make Jekyll process a page without defining variables in the front matter, use: 

---
---

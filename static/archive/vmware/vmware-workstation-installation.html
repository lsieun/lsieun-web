系统要求

官方自VM14版本开始不支持某些老旧电脑硬件，会提示不支持或安装失败, 如遇到请退回12版本。

  • VM16 -> Windows 10 或更高版64位
  • VM15 -> Windows 7 或更高版64位
  • VM12 -> Windows 7 或更高版64位
  • VM10 -> Windows XP 或更高版32位和64位

注意：如果是WinXP或32位系统请用 10.x 版本；11.x 版本之后支持Win7或更高版64位系统。

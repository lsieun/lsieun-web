BIT STRING

Used for bits arrays that are not necessarily multiple of 8 bits.笔记：bit的数量不一定是8的倍数 You can assign meanings to each bit individually. Values have 3 possible forms: binary - '011'B, hex - '6'H, and named - {windowOpen, engineOn}.笔记：有三种不同的表示方式 These are identical values for myStatus in the example.

Sensors ::= BIT STRING { doorOpen(0), windowOpen(1), engineOn(2) } myStatus Sensors ::= {windowOpen, engineOn}

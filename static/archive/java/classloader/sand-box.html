沙箱安全机制

自定义String类，但是在加载自定义String类的时候，会率先使用Bootstrap ClassLoader进行加载；而Bootstrap ClassLoader在加载的过程中会先加载JDK自带的文件（rt.jar包中的java/lang/String.class），报错信息是说没有main方法，就是因为加载的是rt.jar包中的String类。这样可以保证对java核心源代码的保护，这就是“沙箱安全机制”。 

package java.lang;

public class String {
    public static void main(String[] args) {
        System.out.println("Hello String");
    }
}

错误: 在类 java.lang.String 中找不到 main 方法, 请将 main 方法定义为:
   public static void main(String[] args)
否则 JavaFX 应用程序类必须扩展javafx.application.Application

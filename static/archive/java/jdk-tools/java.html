临时改变：在java/javac命令后面加上-Duser.language=en 

$ java -Duser.language=en
$ javac -J-Duser.language=en
$ javap -J-Duser.language=en

$ java -Duser.language=zh
$ javac -J-Duser.language=zh
$ javap -J-Duser.language=zh

永久改变：设置环境变量JAVA_TOOL_OPTIONS，其值为-Duser.language=en。

Java Compiler

要将一个.java文件转换为一个.class文件，则需要用到Java Compiler(javac)。如果说.java文件则相当于\(X\)，.class文件则相当于\(Y\)，那么javac的作用就相当于\(f(x)\)，由它来实现.java文件内容向.class文件内容的映射。

更详细的说，在.java文件中定义的字段、定义的方法等内容一定是在.class文件中得到体现的。

Mapping From language to class file format

Java's architecture arises out of four distinct but interrelated technologies:

  • the Java programming language
  • the Java class file format
  • the Java Application Programming Interface
  • the Java Virtual Machine

The Java Virtual Machine knows nothing of the Java programming language, only of a particular binary format, the class file format.

概括的来说，在Java programming language支持的语言特性必须有相应的Java class file format进行支持。

其中，对于Java programming language来说，它本身是一个抽象的规则，而.java文件是它的一种具体表现形式；对于Java class file format来说，它则是另一个抽象的规则，.class文件则是它的一种具体表现形式。

Summing Up

总结：

  • 第一，对于一个.java通过javac编译成相应的.class文件。换句话说，javac编译过程的本质就是进行“映射”的过程，将.java里的内容映射成.class文件里的内容。
  • 第二，任意一个.java文件都是Java programming language的一种具体表现形式；任意一个.class文件都是Java class file format的一种具体表现形式。
  • 第三，Java programming language中所提供的语言特性，必须在Java class file format层面得到支持，其本质是从Java programming language向Java class file format的语义映射（Mapping）。

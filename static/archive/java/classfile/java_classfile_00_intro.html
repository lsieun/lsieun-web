学习阶段

学习ClassFile的两个阶段：

第一个阶段，有如下几个层次

  1. 元认知
  2. ClassFile基本结构
  3. 从byte角度来解析ClassFile
  4. extensions
  5. 解析bytecode

第二个阶段

  1. Java-Agent
  2. 使用Javassist
  3. 使用ASM

第一个阶段，是针对Java ClassFile的内部原理的解析，注重于“是什么”；第二阶段，是对Java ClassFile原理的运用，注重于“怎么用”。

简单来说，第一个阶段是讲原理，第二个阶段是讲应用。第一个阶段的内容，是第二个阶段内容的基础，用一种形象的说法来讲就是“有九阳神功护体，学什么武功都特别快”。 换句话说，学会了第一个阶段的内容，再学第二阶段的内容就是驾轻就熟（赶着轻载的车走在熟路上）。

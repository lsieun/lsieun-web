Compactness

Compactness is important because it facilitates speedy transmission of class files across networks. In the bytecodes stored in class files, all instructions--except two that deal with table jumping--are aligned on byte boundaries. The total number of opcodes is small enough so that opcodes occupy only one byte. 笔记：在这里opcode占用 1 byte的空间，其目的是compactness；另外，在field或method的descriptor描述中，也体现了compactness，例如用I表示int类型、用Z表示boolean类型。

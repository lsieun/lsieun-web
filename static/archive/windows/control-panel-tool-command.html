Control panel tool Command

TOC Toggle Commentary

This article describes how to run Control Panel tools in Windows by typing a command at a command prompt or in the Open box.

NOTE: If you want to run a command from a command prompt, you must do so from the Windows folder. Also, note that your computer may not have all of the tools listed in this article, as your Windows installation may not include all of these components.

Commands

  • access.cpl: Accessibility Options control. （Win7不支持）
  • appwiz.cpl: Add/Remove Programs control 程序和功能（Win7支持）
  • desk.cpl: Display Properties control 屏幕分辨率（Win7支持）
  • findfast.cpl: FindFast control （Win7不支持）
  • inetcpl.cpl: Internet Properties control Internet属性（Win7支持）
  • intl.cpl: Regional Settings control 区域和语言（Win7支持）
  • joy.cpl: Joystick Properties control 游戏控制器（Win7支持）
  • main.cpl: Mouse Properties control 鼠标属性（Win7支持）
  • wgpocpl.cpl: Microsoft Mail Post Office control （Win7不支持）
  • mlcfg32.cpl: Microsoft Exchange control （Win7不支持）
  • mmsys.cpl: Sound Properties control 声音（Win7支持）
  • modem.cpl: Modem Properties control （Win7不支持）
  • netcpl.cpl: Network Properties control （Win7不支持）
  • password.cpl: Password Properties control （Win7不支持）
  • sticpl.cpl: Scanners and Cameras control （Win7不支持）
  • sysdm.cpl: System Properties control 系统属性（Win7支持）
  • timedate.cpl: Date/Time Properties control 日期和时间（Win7支持）

sysdm.cpl

sysdm.cpl = System Applet for the Control Panel

File location: C:\Windows\System32

sysdm.cpl stands for SYStem Device Manager Control PaneL applet.

设置环境变量快速方法：Win+R唤出“运行”，输入sysdm.cpl，进入系统属性——高级——环境变量。

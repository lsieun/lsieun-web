Pointofix

下载地址：http://www.pointofix.de/

Pointofix是一款由德国人开发的屏幕标记软件，德国人的工匠精神，是出了名的，德国人开发的软件也一样。

Pointofix体积非常小巧，安装包只有1MB大小，使用Pointofix可以直接在屏幕上面写字、画图、标重点。

Pointofix快捷键名称有些奇特，对照如下：

  • Strg = Ctrl
  • Umsch = Shift
  • Ruck = Backspace
  • Entf = Del
